This is a colour palette with all the official colours of the PFC CSKA. There are two primary colour - reb and blue. Those colours using in all the production of the brand, this colours even using as one of the club's nickname. Black, white and gray colours are secondary. Those colours mostly using on the away uniform and in some part as a colours of typography. Yellow colour using only on the logo design, but still this colour is very important for the brand.
For the printing production should be used CMYK color version.
For the digital production (screen, web-sites, mobile app. and etc) should be used RGB color version.
